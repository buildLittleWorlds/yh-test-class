# Youth Horizons AI Researcher Program - Level 2

A 12-week curriculum teaching G7-G11 students to build with AI through guided discovery and hands-on projects.

## Course Structure

| Phase | Weeks | Theme |
|-------|-------|-------|
| Exploration | 1-3 | What's possible? What interests you? |
| Specialization | 4-6 | Go deeper in your chosen track |
| Project | 7-9 | Build something that's yours |
| Showcase | 10-12 | Share what you built |

---

- **Prerequisite:** Level 1 (AI literacy, ethics, prompt engineering basics)
